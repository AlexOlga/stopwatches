## Тестовое задание:
Создать страницу с секундомерами. 

Каждый секундомер должен иметь 3 функции:
1. Запуск
2. Пауза
3. Сброс

Кроме секундомеров на странице должна быть кнопка, которая добавляет новый секундомер в список.
